# kysy

Ask or know (kysy tai tiedä).

Relating answers to files and folders (Finnish: Vastaukset tiedostoihin ja kansioihin).

Prototyping names and concepts:

* ask or know (kysy tai tiedä)
* ellipse (ellipsi)
* file (tiedosto)
* folder (kansio)
